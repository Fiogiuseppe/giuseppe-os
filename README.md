# Giuseppe OS v0.1

A personal operating system for building a life aligned with purpose, freedom, reputation, creativity and financial independence.

## Core idea

A thinking system that helps Giuseppe live his spiritual purpose in practical reality.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Structure

- `/agents` — the six counsellors: CEO 2036, CFO, Strategist, Creative Director, Psychologist, Mentor.
- `/memory` — Giuseppe Brain: purpose, values, rules, projects, finances, patterns.
- `/engine` — decision engine, purpose engine, weekly engine.
- `/app` — Next.js frontend.
- `/docs` — constitution, roadmap, product principles.

This is v0.1: a foundation. The next step is connecting real AI, database, and live data sources.
